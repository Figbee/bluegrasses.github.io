2020-05-24 1 min read

文件相关操作

os 文件操作

Create 新建
Remove 删除
Open 打开
OpenFile 打开文件
OpenFile的使用

//创建并打开
os.OpenFile("file.txt", os.O_CREATE|os.O_RDWR, 0755)
//追加并打开
os.OpenFile("file.txt", os.O_APPEND|os.O_RDWR, 0777)

Stat 返回FileInfo对象

os.Stat("file")

file 文件操作

Read 读取文件内容
ReadAt 读取指定偏移的文件内容
ReadDir 好像只能读取一层

//返回一个文件类型切片
fi, err := f.Readdir(0)

Seek 跳转文件指针

//第一个参数为偏移量,可以为正负数, 第二个参数为偏移基准. 0位从开始,1表示从当前位置,2表示从末尾
file.Seek(7,1)

Trucate 文件截取
Sync 内容从内存中提交到硬盘
Write 文件写入
WriteString 文件以字符串形式写入
Stat 返回FileInfo对象

file.Stat()

fileInfo

Name 文件名
Size 文件大小
